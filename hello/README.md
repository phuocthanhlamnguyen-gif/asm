# Hello, World!

This is a basic "Hello, World!" program written with assembly for Linux `x86_64` as well as for `i686`(aka. 32-bit).

To build the program, run:

```bash
make
```

If you want to get the code, go to:
```bash
code > 32bit-code (if on a 32-bit computer) > hello32.asm
code > 64bit-code (if on a 64-bit computer) > hello.asm
```
For more details, read [Part 1. Introduction](https://github.com/0xAX/asm/blob/master/content/asm_1.md).
